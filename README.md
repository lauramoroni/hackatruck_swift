# Conceitos e Fundamento: Algoritmos e Programação Orientada a Objetos com Swift 
Repositório criado para armazenar os códigos e resumos feitos ao longo do curso de Algoritmos e Programação Orientada a Objetos com Swift. O curso foi realizado pelo [Hackatruck MakerSpace]([https://hackatruck.com.br](https://hackatruck.com.br/area/Home/O+Projeto/Apresentação/)), executado pelo Instituto de Pesquisas Eldorado (e tem como parceiros tecnológicos Cisco, Flex, IBM, em colaboração com a Apple) e oferecido pela [Universidade Federal Rural do Semi-Árido](https://ufersa.edu.br) (UFERSA) de Mossoró-RN.
